# chicago-water-images

Image album for the [Chicago Water System research archive](https://github.com/boswald314/chicago-water-system)
— 611 archival photographs, maps, and diagrams of Chicago's water and sewer infrastructure,
organized by subject and era, served via GitHub Pages to the
[main site's galleries](https://boswald314.github.io/chicago-water-system/gallery/).

Every image's provenance (source page, date, rights status) is in [INDEX.md](INDEX.md);
per-agent download logs are in `_catalogs/`. Rights-restricted items (~33) are deliberately
excluded from this public repo — see the
[attribution & rights policy](https://boswald314.github.io/chicago-water-system/ATTRIBUTION.html).
Rights holders: to request removal or correction, please open an issue.

`thumbs/` mirrors the tree at 480 px for the gallery grids.
