# sources-pages/

Page renders (130 dpi JPEG) from documents in the research archive's `sources/` tree,
generated so that quotes and figures on the site's timeline pages can link directly to
the exact page they come from. One folder per document (folder name = document filename),
one file per page (`pNNNN.jpg`).

Rights: only US-government and public-agency documents are rendered here (EPA, USACE, GAO,
federal courts, Illinois state surveys, MWRD/MSD public records) — public domain or public
records. Commercial sources are quoted on the site without page images.
